# Private Mail Gateway

基于 Next.js 14 构建的安全邮件网关，提供五因子 API 认证、AES-256-GCM 加密以及全面的安全防护。

## 核心功能

- **五因子 API 认证**：API Key + TOTP + Config Key + 请求签名 + Nonce
- **AES-256-GCM 加密**：所有邮箱凭证在 Vercel KV 中加密存储
- **HKDF 密钥派生**：主密钥 → 加密密钥 + 签名密钥
- **管理面板**：三级安全认证（密码 / 密码+TOTP / 密码+TOTP+DNS应急）
- **SSRF 防护**：自定义实现，阻止私有 IP 范围
- **速率限制**：登录、API、接收邮件等多层级限流
- **会话管理**：HttpOnly、Secure、SameSite=Strict Cookie

## 技术栈

- Next.js 14 (App Router)
- TypeScript / Tailwind CSS
- Vercel KV (Redis)
- Nodemailer (SMTP) / ImapFlow (IMAP)
- otplib (TOTP)
